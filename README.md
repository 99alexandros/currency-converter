# Currency Converter

A modern real-time currency converter built with **HTML, CSS, and JavaScript**.

## Features

- Live exchange rates using ExchangeRate API
- 30 popular world currencies with country flags
- Searchable currency dropdowns
- Swap currencies with one click
- 7-day exchange rate chart
- Auto-refresh every 60 seconds
- Conversion history saved with localStorage
- Responsive design for mobile and desktop

## Technologies

- HTML
- CSS
- JavaScript
- Chart.js
- ExchangeRate API
